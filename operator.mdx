# Operator Guide

Raw data layers contain ingested data that has not been transformed and is in
its original file format (for example, JSON or CSV). This data is typically
organized by data source and the date that it was ingested into the raw data
layer's Amazon Simple Storage Service (Amazon S3) bucket

[Learn more](https://docs.aws.amazon.com/prescriptive-guidance/latest/defining-bucket-names-data-lakes/raw-data-layer-naming-structure.html)
about data layers in AWS S3 data lakes.
